Gerar log de ponto Web

Veja como enviar informações técnicas para análise do ponto.

Esse artigo é feito para colaboradores que utilizam a plataforma MarQ. Para realizar o passo-a-passo deve estar logado via web no link.

O log de registros Web é utilizado para uma análise mais técnica sobre as marcações de ponto realizadas, nele é possível identificar todos os passos que o colaborador realizou até o momento da conclusão do registro.

No menu à esquerda (inferior) > “Log de registros”:

Clique em “envio”:

 
Ao clicar nesta opção, o suporte da MarQ. receberá o histórico de todos os passos realizados pelo perfil do colaborador no modo Web.

 


Qualquer dúvida contate o seu gestor/responsável!
