Solicitar Reembolso

Aqui você irá descobrir como o colaborador consegue realizar a solicitação de reembolso

Esse artigo é feito para colaboradores que utilizam a plataforma MarQ. Para realizar o passo-a-passo deve estar logado via app MarQPonto ou via web no link.

No app, clique em Menu > Reembolso:

 

Clique em "Solicitar Reembolso" > insira todas as informações do reembolso (podendo incluir um arquivo ou até uma foto direto do app > Solicitar reembolso:

 

Prontinho, agora é só esperar o administrador aceitar o reembolso!

Via web o colaborador também consegue realizar a solicitação de reembolso, clicando no menu à esquerda "Reembolsos" > Clique em "Solicitar Reembolso" no canto superior direito:

Após isso é só colocar as informações sobre o reembolso e confirmar:

 

Pronto, agora é só aguardar o administrador aceitar o reembolso!

 

Qualquer dúvida contate o seu gestor/responsável!
