Marcar ponto via Web

Veja como marcar seu ponto via Web na MarQ.

Esse artigo é feito para colaboradores que utilizam a plataforma MarQ. Para realizar o passo-a-passo deve estar logado via web no link.

No menu à esquerda > “marcar ponto”:

 

Ao selecionar esta opção, você é redirecionado para a tela de ponto, onde além de marcar, pode validar algumas informações, tais como: o ultimo ponto registrado, a quantidade de horas trabalhadas e a virada do dia para aquele turno.

 

 

 

Mas para registrar, é preciso permitir a localização do dispositivo, assim como a permissão para uso de webcam - para os casos onde a empresa exija o registro do ponto com foto:
