Sincronize pontos pendentes via App

Veja como é fácil sincronizar os pontos offline!

Esse artigo é feito para colaboradores que utilizam a plataforma MarQ. Para realizar o passo a passo deve estar logado via app MarQPonto.

Antes, confirme com a sua gestão se há uma autorização para realizar a marcação de ponto offline.

A MarQ. permite que você possa registrar o ponto de forma offline (sem wifi e sem redes móveis) onde esse ponto é armazenado no próprio aplicativo e fica pendente para sincronização.
​
A sincronização é o processo de identificar o registro e enviar para o servidor da MarQ. possibilitando que tanto você quanto o seu gestor/administrador veja esse ponto marcado.
​
​Importante: Tome cuidado ao desinstalar o aplicativo, porque se tiver pontos pendentes para sincronização eles serão excluídos junto com o aplicativo e não há forma de recuperar.

Para fazer a sincronização você precisa ter sinal de internet. Siga este passo a passo:
