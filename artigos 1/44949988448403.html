Edite dados do perfil via App

Como editar os dados de um perfil com o acesso App na MarQPonto.

Esse artigo é feito para colaboradores que utilizam a plataforma MarQ. Para realizar o passo-a-passo deve estar logado via app MarQPonto.

Para realizar a edição dos dados do perfil pelo App MarQPonto, siga:

Clique em Menu no canto inferior direito da tela de início:

 

Logo em seguida, clique em “Ver perfil”:

 

 

Edite as informações e salve!

 

Qualquer dúvida contate o seu gestor/responsável!
